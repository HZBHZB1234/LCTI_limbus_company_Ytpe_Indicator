# EGO特质测试

一个基于七种特质理论的趣味性格测试网页，通过11道题目探索你的内在EGO特质。

## ✨ 功能特点

- **快速测试**：仅需2-3分钟完成11道题目
- **七种特质**：暴食、暴怒、傲慢、欲望、忧郁、嫉妒
- **可视化结果**：特质分布柱状图 + 显著特质标记
- **人格类型判定**：根据答题结果自动判定主导人格类型（包括复合型与隐藏彩蛋）
- **响应式设计**：适配手机、平板、电脑等不同屏幕尺寸
- **纯前端实现**：无需后端服务器，开箱即用

## 🛠 技术栈

- HTML5
- CSS3 (Flex/Grid布局、CSS变量、动画)
- JavaScript (ES6+)
- Font Awesome 6.4.0 (图标库)

## 🚀 使用方法

1. **下载或克隆项目**到本地
2. 确保 `images/` 文件夹内放置了对应的人格类型图片（图片名称参考 `script.js` 中的 `PERSONALITY_IMAGES` 映射）
3. 直接双击 `index.html` 或在浏览器中打开该文件
4. 点击「开始测试」逐题作答，最后查看结果

> ⚠️ 注意：由于使用了 `fetch` 加载 `questions.json`，建议使用本地服务器（如 VS Code Live Server）运行以避免跨域问题。部分浏览器直接打开文件可能无法加载 JSON 数据。

## 🧩 题目与特质机制

- 每道题的选项对应一个数值，该数值会通过模运算分配到不同特质计分中
- 最终根据各特质出现频率判定显著特质（超过最大可能次数的1/3）
- 总分与特质组合共同决定人格类型描述

## 🎨 自定义扩展

- **修改题目**：编辑 `questions.json` 文件即可增删改题目或选项
- **调整特质权重**：修改 `script.js` 中的 `TRAITS` 对象及计分逻辑
- **更换配图**：替换 `images/` 文件夹中的图片，并在 `PERSONALITY_IMAGES` 中更新路径
